# assuring-and-testing-lab12

Энэхүү репозитор нь лифт хяналтын системийн төгсгөлөг төлөвийн машин (FSM) загвар болон түүнд хамаарах туршилтын кейс, сценариудыг баримтжуулсан байна.

Чухал файлууд

- `fsm_definition.md` — FSM-ын албан ёсны тодорхойлолт: төлөвүүд, үзэгдлүүд, шилжилтүүд болон үнэн зөв байдлын шинжүүд.
- `fsm_tests_and_scenarios.md` — лифт FSM-ийн туршилтын кейс болон эцсийн (end-to-end) сценариуд.
- `Untitled Diagram.drawio` — FSM-ийг дүрслэн харуулсан диаграм.

Хэрхэн ашиглах

- `fsm_definition.md` файлыг уншиж, лифтийн төлөвүүд болон боломжит шилжилтүүдийг ойлгоно.
- `fsm_tests_and_scenarios.md` файлыг ашиглан гар аргаар турших эсвэл автоматжуулсан туршилтуудыг хэрэгжүүлэх сценариудыг ашиглана.
